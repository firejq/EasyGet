# EasyGet

A download tool supporting multi-thread transfer and breakpoint restoration.

## Feature

- [x] 自动 / 手动指定线程数量
- [ ] 当前速度、进度可视化
- [x] 支持断点下载
<!-- - [ ] 支持 ipv6，当 ipv4 与 ipv6 并存时优先使用 ipv6 -->

## Usage

```
java -jar EasyGet.jar targetUrl localPath [threadNumber]
```
